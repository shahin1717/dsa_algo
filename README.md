# DSA Practice & Learning Hub 🧠💻

Collection of fundamental **Data Structures & Algorithms** implementations + some mixed practice problems from competitive programming platforms (mainly **Codeforces** & **AtCoder**).

Perfect for learning, revising before interviews, or preparing for CP contests.

## 🗂 Current Focus Areas

| #  | Category           | Files                              | Status     |
|----|--------------------|------------------------------------|------------|
| 1  | Graphs             | BFS, DFS, Dijkstra                 | ✅ Core    |
| 2  | Search & Prefix    | Binary Search, Prefix Sum          | ✅ Core    |
| 3  | Dynamic Programming| Basics + Knapsack family           | ✅ Started |
| 4  | Mixed CP Practice  | Codeforces & AtCoder problems      | 🛠 In progress |
